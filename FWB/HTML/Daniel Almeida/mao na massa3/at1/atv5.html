O que se aprende:

  1. HTML: Htmld ou HyperText Markup Language, é uma linguagem de marcação usada para criar e estruturar páginas da web, utilizando tags para definir o conteúdo e sua apresentação.

    É a base da construção de páginas web, permitindo incluir texto, imagens, links e outros elementos.

  2. JavaScrpit: Uma linguagem de programação utilizada para criar interatividade em páginas da web, sendo capaz de manipular elementos, validar formulários, realizar requisições a servidores e criar efeitos visuais.

  3. CSS, ou Cascading Style Sheets, é uma linguagem de estilo usada para controlar a apresentação e o layout de páginas da web.

    Com o CSS, os desenvolvedores podem definir cores, fontes, margens, espaçamento e outros aspectos visuais de forma eficiente e consistente.

    Ele permite criar layouts complexos, ajustar o tamanho e a posição dos elementos, criar animações e tornar uma página responsiva para diferentes dispositivos.

    É fundamental no desenvolvimento web para separar o conteúdo da apresentação, facilitando a manutenção e o design consistente das páginas da web.
