Curso - Python Aplicado às Finanças

    Publico

  • Profissionais, estudantes ou interessados na área de TI/Financeira que tenham feito o Módulo 1
    do curso no semestre anterior (2023/2) ou tenha conhecimento básico sobre o tema.

    • objetivo do curso

  • Este curso inicia trabalhando algumas das mais importantes bibliotecas em Python para Análise de Dados.
    Utilizaremos Jupyter Notebook e Jupyter Lab via Anaconda ou Google CoLab.
    Construção de dados e análise analítica para tomada de decisão no Mercado Financeiro

    • conteudo do curso

  • Aprender como aplicar NumPy, Pandas, Matplotlib e Seaborn para análise e plotagem dos gráficos.
    Aplicar técnica de simulação de Monte Carlo para otimizar alocação de portifolio.
    Entender dados fundamentais do Stock Market, tais como CFC, Revenue, and EPS.
    Entender retornos cumulativos e retornos de médias diárias em ações.
    Aplicar QuantConnect's para automatizar operações - Shorting and Leveraging; Ordem Systems.
    Usar algoritmo para negociar contratos de derivativos. Conduzir pesquisa em QuantConnect.
