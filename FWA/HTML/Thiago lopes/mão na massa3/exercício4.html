    • O que são tags HTML?

    Tags HTML são instruções simples que informam ao navegador como formatar texto.
    Você pode usar tags para formatar itálico, quebras de linha, objetos, marcadores e muito mais.
    Essas tags residem no HTML (ou linguagem de marcação de hipertexto)
    de cada página da web. Simplificando, HTML é a linguagem das páginas da web.
